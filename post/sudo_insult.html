Having sudo insult you

To change sudo setting you have to use the visudo command. If you are not too comfortable with vi to edit files like me you may want to invoke visudo with nano. 

$ sudo EDITOR=nano visudo

To turn on insults add the line

Defaults insults

Below the other defaults and save the file.

Now whenever you you type your password incorrectly sudo will insult you!

The insults are super lame and most make no sense, but it's still funny. Some examples:

Harm can come to a young lad like that!
I have been called worse.
stty: unknown mode: doofus
This mission is too important for me to allow you to jeopardize it.
