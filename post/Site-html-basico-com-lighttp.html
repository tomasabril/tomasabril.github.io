Site html básico com lighttp

Para criar o site em si editei manualmente um .html e utilizei o sakura pra deixá-lo um pouco mais bonito.

https://github.com/oxalorg/sakura

Como servidor usei o Lighttpd.

https://wiki.archlinux.org/index.php/Lighttpd

Estou instalando no raspberry pi 3 Raspbian Stretch, o pacote está nos repositórios oficiais.

$ sudo aptitude install lighttpd lighttdp-doc

As configurações ficam no arquivo

/etc/lighttpd/lighttpd.conf

Após modificá-las é possível checar por erros com o comando

$ lighttpd -t -f /etc/lighttpd/lighttpd.conf

Por padrão o lighttpd procura por um arquivo na pasta

/var/www/html/

com o nome

index.html

O seu site pode ser acessado apontando seu navegador para seu

ip:porta

sendo a porta 80 o padrão.

Para executar o servidor em todo boot.

$ systemctl enable lighttpd.service

https://wiki.archlinux.org/index.php/Systemd#Using_units
